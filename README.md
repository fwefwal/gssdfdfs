## 1. Общие сведения

### 1.1. Назначение сайта
Сайт кофейни «Coffee Haven» предназначен для:
- Представления ассортимента кофе, десертов и других позиций меню
- Приёма онлайн-заказов с доставкой и самовывозом
- Информирования о расположении кофейни, режиме работы и актуальных акциях
- Сбора контактных данных клиентов для email- и SMS-рассылок
- Увеличения посещаемости кофейни на **8 %** в первые 3 месяца после запуска

### 1.2. Целевая аудитория
- Возраст: 16–30 лет
- Пол: 60 % женщины, 40 % мужчины
- География: г. Санкт-Петербург, радиус 5 км от кофейни
- Интересы: качественный кофе, уютные места, быстрые онлайн-заказы

### 1.3. Тип сайта
Адаптивный лендинг + полноценный интернет-магазин (корзина, онлайн-оплата)

## 4. Технические требования

### 4.1. Дизайн
- Стиль: минимализм, тёплые тона (бежевый, коричневый, кремовый)
- Шрифты: заголовки — **Playfair Display**, текст — **Roboto**
- 100 % адаптивность (мобильные, планшеты, десктоп)
- Анимации: плавные fade-in при скролле, hover-эффекты на карточках товаров
- Иконки: Font Awesome

### 4.2. Технологии и производительность
- Хостинг: Timeweb или Beget (SSD, PHP 8.1+)
- База данных: MySQL 5.7+
- Frontend: HTML5, CSS3 (SCSS), JavaScript (React опционально)
- Скорость загрузки: **< 3 сек**, Google PageSpeed Insights **90+** баллов

### 4.3. Интеграции
(в ТЗ не перечислены конкретные сервисы, предполагаются:)
- Платёжные системы (ЮKassa, CloudPayments и др.)
- Службы доставки (при необходимости)
- Google Analytics, Яндекс.Метрика
- Сервисы email-рассылок

### 4.5. SEO и безопасность
- Базовая SEO-оптимизация: мета-теги, alt-тексты, sitemap.xml, robots.txt
- SSL-сертификат (HTTPS)
- Защита от SQL-инъекций и XSS
- reCAPTCHA на формах
- Подготовка к PWA (возможность установки на телефон)
