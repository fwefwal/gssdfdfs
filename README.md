2.2 Функциональные требования  

| ID | Требование |
|----|------------|
| F-01 | Регистрация/авторизация пользователей (email + пароль, OAuth Google/Yandex). |
| F-02 | Добавление объекта недвижимости (фото, адрес, характеристики, цена, статус). |
| F-03 | Редактирование/удаление собственных объектов (только владельцем или модератором). |
| F-04 | Расширенный поиск с фильтрами: тип, цена, площадь, район, количество комнат, этаж, год постройки, наличие балкона/парковки и др. |
| F-05 | Сохранение поисковых запросов и подписка на новые объекты по фильтру (email-уведомления). |
| F-06 | Отображение объектов на интерактивной карте (Yandex Maps / OpenStreetMap). |
| F-07 | Модерация объявлений администратором (одобрение/отклонение). |
| F-08 | Импорт объектов из CSV/XML (администратор). |
| F-09 | Экспорт результатов поиска в CSV/PDF. |
| F-10 | Генерация аналитических отчётов (по районам, типам, ценам) для администратора. |
| F-11 | Управление ролями пользователей (пользователь, риелтор, администратор). |
| F-12 | Возможность отмечать объект как «Избранное». |
| F-13 | Отправка заявки на просмотр объекта (форма обратной связи). |
