### 3.1 Требования к интерфейсу  
- Веб-интерфейс, совместимый с последними версиями **Chrome, Firefox, Safari, Edge**.  
- Адаптивный дизайн (мобильные устройства ≥ 320 px).  
- Соответствие WCAG 2.1 уровня A.  

### 3.2 Требования к производительности  
| Метрика | Требование |
|---------|------------|
| Время отклика поиска | ≤ 1.5 с при 100 000 объектов |
| Время загрузки карточки объекта | ≤ 2 с |
| Одновременных пользователей | ≥ 500 (без деградации > 10 %) |

### 3.3 Требования к безопасности  
- HTTPS (TLS 1.3).  
- Хэширование паролей (bcrypt).  
- Защита от SQL-инъекций, XSS, CSRF.  
- Ролевая модель доступа (RBAC).  
- Аудит действий администраторов.  

### 3.4 Эксплуатационные требования  
- ОС сервера: Linux (Ubuntu 22.04 LTS или новее).  
- Веб-сервер: Nginx.  
- БД: PostgreSQL 15+.  
- Хостинг: облачная платформа с автоскейлингом.  

### 3.5 Политические и юридические требования  
- Соответствие **GDPR** (возможность удаления персональных данных по запросу).  
- Хранение данных на территории ЕС (для пользователей из NL).  
- Отсутствие дискриминационных фильтров.  

### 3.6 Другие ограничения  
- Максимальный размер загружаемого фото — 5 МБ, форматы JPEG/PNG.  
- Общее количество фото на объект — ≤ 20.
